## 1 背景介绍

### 1.1 项目来源
数控加工仿真产品线需求

### 1.2 应用领域与范围
五轴数控加工中心球头刀切削过程的几何仿真

### 1.3 时间要求
2个月内交付核心算法原型

### 1.4 客户主要需求
现有切削仿真采用体素法，计算量大、精度低。客户需要一种基于自适应四叉树的 tri-dexel 表示法来高效精确地模拟球头刀切削过程。

关键痛点：
1. 现有方法计算时间 > 10分钟，无法接受
2. 曲面交界处精度不足（±0.1mm）
3. 复杂型腔加工时内存占用过高

### 1.5 工艺介绍
球头刀（Ball-end Mill）三轴/五轴数控加工
- 刀具：半径 R 的半球形刀头
- 运动：直线插补 + 旋转
- 材料去除：刀具包络面与工件求交

### 1.6 竞争度分析
Vericut、NCSimul 等商用软件，价格昂贵

## 3 项目需求

### 3.1 差异价值分析
比体素法快10倍，比传统dexel法精度高

### 3.2 需求说明
构建基于 adaptive quadtree + tri-dexel 的球头刀切削仿真核心算法

### 3.3 需求列表

| 序号 | 名称 | 描述 | 优先级别 |
|:---:|:---|:---|:---:|
| 1 | 自适应四叉树空间分割 | 基于切削区域密度的动态网格细分 | 关键 |
| 2 | Tri-Dexel几何表示 | 三方向深度像素表示工件几何 | 关键 |
| 3 | 球头刀包络计算 | 刀具扫掠体与dexel求交算法 | 关键 |
| 4 | 材料去除仿真 | 更新tri-dexel表示模拟切削 | 高 |
| 5 | 精度与性能基准 | 对比测试与精度验证 | 中 |
